Installation
------------
You can install directly from the source:

.. code-block:: console

    $ git clone https://github.com/jfeatherstone/spatialstats
    $ cd spatialstats
    $ pip install .

Development
-----------

To run the test suite, we recommend using pytest:

.. code-block:: console

    python -m pytest testing/ -v
